pml
===
Please run code and server from a convenient Terminal or Unix Shell(Bash)
Then proceed to 
	http://localhost:9000 
in your favorite browser.


Example input:
 
<html>
<h1>Hello There</h1>
<p>
This is an example of a pml file
</p>
<pml>
    def f():
        return "<h2>Good Bye</h2>"
 
    pml = f()
</pml>
</html>
 
 
Example output:
 
<html>
<h1>Hello There</h1>
<p>
This is an example of a pml file
</p>
<h2>Good Bye</h2>
</html>
